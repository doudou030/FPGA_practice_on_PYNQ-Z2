# FPGA_practice_on_PYNQ-Z2
## 說明
大三的專題code:FPGA實作手寫數字辨識小遊戲

主要目的是在練習FPGA板子的使用及GUI的結合


main.py在自己的laptop跑

而有一個on_pynq的資料夾裡有一個final.py則是需要丟到pynq上跑

在實作的時候需要先執行PYNQ-Z2上的final.py再去執行laptop這邊的main.py
